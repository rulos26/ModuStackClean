# 🔧 SOLUCIÓN AL ERROR DE UNICODE - INFO_SISTEMA.PY

## ❌ Problema Original:
```
UnicodeEncodeError: 'charmap' codec can't encode character '\U0001f527' in position 0: character maps to <undefined>
```

## ✅ Solución Implementada:

### 🔍 **Causa del Error:**
- Windows usa la codificación `cp1252` por defecto en la consola
- Los emojis (caracteres Unicode) no pueden ser mostrados en esta codificación
- El script intentaba mostrar emojis como 🔧, 📊, ✅, etc.

### 🛠️ **Solución Aplicada:**

1. **Configuración automática de UTF-8** para Windows
2. **Eliminación de emojis** en la salida de consola
3. **Manejo de errores** para evitar fallos

### 📝 **Cambios Realizados:**

```python
def main():
    """Función principal"""
    # Configurar codificación para Windows
    import sys
    import os
    
    # Intentar configurar UTF-8 para Windows
    if sys.platform.startswith('win'):
        try:
            # Configurar la consola para UTF-8
            os.system('chcp 65001 >nul 2>&1')
            # Configurar stdout para UTF-8
            sys.stdout.reconfigure(encoding='utf-8')
        except:
            pass  # Si falla, continuar sin emojis
    
    print("Obteniendo información del sistema...")  # Sin emoji
    # ... resto del código sin emojis
```
